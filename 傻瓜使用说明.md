# 高岛断易起卦录：傻瓜使用说明

> 不懂《易经》也可以先用。你只需要准备一个明确的问题，然后按照下面的顺序操作。

## 一句话版本

打开网页 → 写问题 → 点击“一键起卦” → 查看卦象 → 点击“导出卦象” → 自行保存或发给你选择的 AI。

---

## 第一步：打开工具

双击本目录中的 `index.html`。

打开以后，不需要登录，也不需要联网。

## 第二步：写清楚你要问什么

在“所占何事”里，只问一件事。

推荐写法：

```text
在未来三个月内，我是否应该继续执行当前的交易方案？
```

不推荐写法：

```text
我的事业、感情、财运以后会怎么样？
```

因为后者一次问了很多件事，之后很难判断卦象究竟在回答哪一件。

需要时间范围时，直接写进问题，例如“未来三个月”“本月底以前”。

## 第三步：点击“一键起卦”

确认问题后点击一次“一键起卦”。页面会自动记录起卦时间、模拟六次三钱法，并从初爻到上爻生成完整卦象。

同一个问题不要因为不喜欢结果而重复点击。页面成卦后会锁住按钮。

电脑使用浏览器随机数模拟六次三钱法，不会读取你的个人资料。

## 第四步：看结果

六爻完成以后，右边会自动出现：

- **本卦**：这次占问最初形成的卦。
- **动爻**：6 和 9 对应的变化位置，是分析时的重要焦点。
- **之卦**：动爻变化后形成的卦。
- **互卦**：从本卦中间四爻重新组合出来的卦，可作为内部结构参考。
- **原始数列**：六次结果，顺序永远是自下而上。

页面只负责算卦象，不会自动替你下结论。

## 第五步：导出卦象

页面下面只有一个“导出卦象”按钮。点击后，Markdown 文档只保留问题和卦象；你可以自行保存，也可以发送给任意 AI。

工具只负责导出问题和卦象，不附带分析结论、提示词或自动接管规则。

## 第六步：把导出文件放到正确位置

浏览器导出的文件通常先进入 Windows 的“下载”文件夹。

请把它移动到本目录的 `卦象/` 文件夹。

如果这个文件夹还不存在，可以自行新建。公开仓库默认忽略 `卦象/` 和 `解读/`，防止个人问题与复盘被误上传；但提交到 GitHub 前仍要检查文件列表。

建议同时保留：

```text
2026-07-18_是否继续当前交易方案.md
```

## 第七步：自行选择是否交给 AI

你可以把导出的 Markdown 直接发给 Gemini、DeepSeek、Codex 或其他 AI。例如只需说：

```text
请分析这份高岛断易卦象，并说明你的依据和不确定性。
```

是否同时提供 ZiWei、当前现状或其他资料，完全由你当次决定。页面和导出文件不会替你做这个选择。

发送前请先检查问题里是否包含姓名、账号、联系方式、详细住址、证件号码或他人隐私。只提供分析所必需的信息。

## 第八步：事情结束后自行复盘

不要修改 `卦象/` 中的原始问题和卦象。请在 `解读/` 中另建一份文件，分别记录：

- 实际发生了什么；
- 哪些判断吻合；
- 哪些判断冲突；
- 哪些地方信息不足；
- 有没有因为解释太宽泛，导致任何结果都能往里面套。

原始卦象始终保持不变，这样以后可以核对起卦时究竟是什么结果。

---

## 最常见的错误

1. **一次问很多件事**：拆成不同问题，在不同时间分别起卦。
2. **六爻顺序填反**：第一次永远是最下面的初爻，第六次才是上爻。
3. **同一个问题反复起卦**：因为不喜欢第一次结果而马上重起，会让卦象失去分析价值。
4. **事情发生后修改原问题或卦象**：后续结果应另写文件，不要改动原始卦象。
5. **只记准的，不记错的**：这样永远无法判断方法是否真的有帮助。
6. **把 AI 生成内容当成原典**：没有书名、版本和页码，就不能写成《高岛易断》原文。
7. **导出后找不到文件**：浏览器通常会把 Markdown 放在 Windows 下载目录。

## 我建议你的固定用法

正式问题采用下面这个流程即可：

```text
一个明确问题
→ 需要时把时间范围写进问题
→ 点击一次“一键起卦”
→ 导出 Markdown
→ 自行保存或发给你选择的 AI
```
