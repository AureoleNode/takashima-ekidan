# 更新记录

## v1.1.0 — 2026-07-23

### 新增

- 结果页新增“复制卦象”，可直接粘贴到 AI 对话；
- 在线版优先使用安全剪贴板 API，离线版提供兼容复制方式；
- 首页显示当前版本号。

### 改进

- 首页重构为统一的起卦工作台；
- 重新整理标题、说明、输入框和主按钮的对齐与视觉层级；
- 减少大块空白，优化桌面和手机布局；
- 结果页将“复制卦象”设为主要操作，“导出卦象”保留用于归档；
- 更新 README、隐私说明和傻瓜使用说明。

## v1.0.0 — 2026-07-23

- 首个公开版本；
- 支持一键模拟六次三钱法；
- 计算本卦、动爻、之卦和互卦；
- 支持 Markdown 导出、日间模式和夜间模式；
- 支持 GitHub Pages 在线使用。
