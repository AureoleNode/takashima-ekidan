# 隐私与本地数据说明

## 项目如何处理数据

高岛断易 · 起卦录是一个静态网页。它没有后端服务，不需要账号，也不会主动发送网络请求。

- 你输入的问题仅保存在当前页面的内存中；
- 起卦结果仅保存在当前页面的内存中；
- 页面关闭或刷新后，上述内容不会由本项目自动恢复；
- 浏览器本地存储只记录日间或夜间主题偏好；
- 只有点击“导出卦象”后，浏览器才会在本机生成 Markdown 文件。

## 在线版与离线版的区别

在线版由 GitHub Pages 托管。打开页面时，浏览器需要向 GitHub 请求 HTML、CSS 和 JavaScript，因此 GitHub 可能按照其服务规则处理常规访问日志，例如 IP 地址、浏览器类型、请求时间和所访问的文件。

页面加载完成后：

- 问题、随机结果和卦象计算都在浏览器中完成；
- 本项目没有把问题发送给 GitHub 或其他第三方的代码；
- 本项目没有广告、分析统计、第三方字体或外部脚本；
- 导出操作只在你的设备上生成文件。

如果问题本身高度敏感，建议下载仓库后断网打开 `index.html`。使用公共或共享电脑时，不要留下导出的文件。

## 使用者需要注意

导出的文件可能包含问题、日期和卦象。问题中不要写入身份证号、银行卡号、账号密码、详细住址、未公开联系方式、他人隐私或其他不必要的敏感信息。

本仓库默认通过 `.gitignore` 排除：

```text
卦象/
解读/
```

这能降低个人记录被误提交的风险，但不能替代提交前检查。每次发布到公开仓库前，仍应运行 `git status` 和 `git diff --cached`，确认没有敏感文件。
